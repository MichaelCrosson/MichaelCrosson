### LightRAG Studio
**Local-first document search and knowledge graph platform**

A streamlined local implementation of LightRAG for intelligent document analysis and querying.  
No cloud dependencies. Everything runs locally.

**Key features**
- Hybrid RAG system with local, global, and mixed query modes  
- Knowledge graph visualization of document relationships  
- Interactive web frontend for document management and search  
- Local-first architecture with no external services  
- Multi-format document upload (PDF, DOCX, TXT)

**Machine learning analysis of what makes tech videos successful**

A research-driven project to identify which features drive engagement in YouTube tech review videos.
